# Rock Paper Scissors Online

Браузерная игра 1v1 с комнатами (создание и подключение).

## Что уже реализовано

- Создание комнаты и подключение по коду.
- Игра в реальном времени через WebSocket.
- Матч до `N` побед (задается при создании комнаты).
- Кнопка реванша после завершения матча (нужно подтверждение обоих игроков).
- Автоудаление неактивных комнат (если нет подключений 10 минут).

## Локальный запуск

```bash
python -m venv .venv
.venv\\Scripts\\activate
pip install -r requirements.txt
python app.py
```

Открой: http://localhost:8000

## Как играть

1. Игрок 1 вводит ник, выбирает `до скольки побед`, нажимает `Создать комнату`.
2. Передает код комнаты игроку 2.
3. Игрок 2 вводит ник, код и нажимает `Подключиться`.
4. Оба делают ходы; после конца матча можно нажать `Реванш`.

## Чтобы играть с друзьями через интернет

Нужно задеплоить приложение и отправить друзьям публичный URL.

### Вариант 1: Render

1. Загрузи проект в GitHub.
2. В Render выбери `New +` -> `Blueprint`.
3. Подключи репозиторий; Render подхватит [render.yaml](./render.yaml).
4. После деплоя получишь URL вида `https://<name>.onrender.com`.

### Вариант 2: Railway

1. Загрузи проект в GitHub.
2. В Railway создай `New Project` -> `Deploy from GitHub repo`.
3. Railway использует [railway.json](./railway.json) и `Procfile`.
4. После деплоя получишь публичный домен проекта.

## Технические детали

- Бэкенд: FastAPI + Uvicorn.
- Фронтенд: один `static/index.html` (vanilla JS).
- Состояние комнат хранится в памяти процесса (без БД).
- В комнате максимум 2 игрока.
