# Cube Tanks Online

Мультиплеерная браузерная игра с комнатами: топ-даун "танчики".

## Что реализовано

- Комнаты с созданием/подключением по коду.
- Безлимитное количество игроков в комнате.
- Движение по X/Y (без прыжка).
- Стрельба в сторону движения.
- Увеличенное игровое поле.
- Когда в живых остается 1 игрок, показывается победитель.
- Автоперезапуск раунда для всех онлайн-игроков.

## Управление

- `WASD` или стрелки: движение
- `Space`: стрельба

## Локальный запуск

```bash
python -m venv .venv
.venv\\Scripts\\activate
pip install -r requirements.txt
python app.py
```

Открой: http://localhost:8000
