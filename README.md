# Cube Tanks Online

Мультиплеерная браузерная игра с комнатами: топ-даун "танчики".

## Что реализовано

- Комнаты с созданием/подключением по коду.
- Безлимитное количество игроков в комнате.
- Движение по X/Y (без прыжка), стрельба в сторону движения.
- Увеличенное игровое поле.
- Раунд заканчивается, когда в живых остается 1 онлайн-игрок (или ничья при одновременном выбивании).
- После конца раунда автоперезапуск для всех онлайн-игроков.
- Отключившиеся игроки удаляются из комнаты и не отображаются офлайн-списком.

## Настройки комнаты (задает создатель)

При создании комнаты можно настроить:

- скорость танка
- скорость пули
- перезарядку (cooldown)
- урон
- HP
- задержку перезапуска раунда

## Управление

- `WASD` или стрелки: движение
- `Space`: стрельба

## Локальный запуск

```bash
python -m venv .venv
.venv\\Scripts\\activate
pip install -r requirements.txt
python app.py
```

Открой: http://localhost:8000
