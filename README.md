# Cube Tanks Online

Мультиплеерная браузерная игра с комнатами: топ-даун "танчики".

## Что реализовано

- Комнаты с созданием/подключением по коду.
- Безлимитное количество игроков в комнате.
- Движение по X/Y, стрельба в сторону движения.
- Увеличенное игровое поле.
- Раунд заканчивается, когда среди активных участников остается 1 живой.
- Автоперезапуск раунда после таймера.

## Важные изменения по сети

- При кратком обрыве сети игрок не удаляется и матч не перезапускается.
- Игрок может переподключиться и продолжить с тем же `player_id`.
- В браузере включено авто-переподключение.

## Настройки в реальном времени (для создателя комнаты)

Во время матча создатель может менять ползунками:

- скорость танка
- скорость пули
- перезарядку
- урон
- HP
- задержку перезапуска

## Управление

- `WASD` или стрелки: движение
- `Space`: стрельба

## Локальный запуск

```bash
python -m venv .venv
.venv\\Scripts\\activate
pip install -r requirements.txt
python app.py
```

Открой: http://localhost:8000
