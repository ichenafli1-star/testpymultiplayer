# Cube Shooter Online

Онлайн 1v1 шутер с комнатами: кубики бегают, прыгают и стреляют.

## Что есть

- Создание комнаты и подключение по коду.
- Игра в реальном времени через WebSocket.
- Когда в живых остается один игрок, показывается победитель.
- Автоперезапуск раунда для всех онлайн-игроков в комнате.
- Автоудаление неактивных комнат (10 минут без подключений).

## Локальный запуск

```bash
python -m venv .venv
.venv\\Scripts\\activate
pip install -r requirements.txt
python app.py
```

Открой: http://localhost:8000

## Управление

- `A/D` или `←/→`: движение
- `W` или `↑`: прыжок
- `Space`: стрельба

## Деплой

- Render: используй `render.yaml`
- Railway: используй `railway.json` / `Procfile`
